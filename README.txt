== Description ==
Woo Extra Cart Fee plugin allows merchant to add extra fee based on product as well as based on product variations.

= Plugin Functionality: =
* Add extra fee based on product
* Add extra fee based on product variations

== Installation ==

* Download the plugin
* Upload the folder "woo-extra-cart-fee" to wp-content/plugins (or upload a zip through the WordPress admin)
* Activate and enjoy!
